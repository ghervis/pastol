Help

Why
You are on a different tab of your browser or alt-tabbed from the game. You are on your phone with Google Chrome opened and you want to get notified about some in-game events.
Custom Notifications
Click the Add add button to create a custom notification. This custom notification can be scheduled with a specific date or time. This notification can be repeated on specific days of the week. Custom notifications can be deleted by clicking on the trash icon delete. Custom notifications can be disabled by clicking on the alarm icon alarm.
Calendar Events
Input an iCalendar URL to populate Pastol with your calendar events. These events will be used to create notifications. These events can be disabled by clicking on the calendar icon event. Calendar events can only be deleted when that item is removed from the source calendar. These events can will have the resynced button sync enabled when the event text is changed locally or when that event is deleted from the sourced iCalendar.
Five Minutes Early Notification
Calendar Events and Custom Notifications can be triggered five minutes early by clicking on the clock watch icon timer_5. This will trigger an additional notification five minutes before the actual notification.
Time of Day
These will be displayed as either Day brightness_5, or Night dark_mode depending on your set Server Region.
Clicking on this section will trigger a one-time notification when the current time of day switches to another time of day (day to night or vice-versa).
Weather
These will be displayed as either Normal sentiment_satisfied, or Rain rainy depending on your set Server Region.
Clicking on this section will trigger a one-time notification when the current weather switches to another weather (normal to rain or vice-versa).
Voice
Select your desired voice in the Settings menu. I was hoping to add a custom voice but browser limitation said no.
Voice Preview
Click the Preview icon record_voice_over to hear a preview of the notification's text.
Reset
If all else fails, please Reset the data in the Settings menu.
